## Технологии и контекст

- **SpeechKit STT** (Yandex Cloud) — speech-to-text: аудиофайл → текст. MVP использует **синхронный** REST endpoint (не streaming, не async jobs).
- **Формат Telegram voice** — OGG Opus. TG-01 скачивает файл; твой клиент читает bytes с диска и шлёт в API с `format=oggopus`, `lang=ru-RU`.
- **Авторизация** — `Authorization: Api-Key <YANDEX_API_KEY>`, folder — `YANDEX_FOLDER_ID`. Ключи только из Django settings / `.env`.
- **Где в пайплайне:** CORE-07 → STT → parser. Handler и лимиты duration/size — не здесь.
- Документация: `tz/VOICE_PIPELINES.md` § SpeechKit STT, § Ошибки и ретраи.

## Перед началом

- Yandex Cloud: `YANDEX_API_KEY`, `YANDEX_FOLDER_ID`, роль `ai.speechkit-stt.user`
- Ключи в `.env`, placeholder в `.env.example`

## Границы тикета

**Входит:** синхронный STT: OGG Opus файл → русский текст.

**Не входит:** скачивание voice (TG-01), лимиты (TG-01, CORE-07), parser (AI-03/06), Telegram.

## Что сделать

1. Интерфейс STT: путь к аудиофайлу → `STTResult` (см. VOICE_PIPELINES).
2. Реализация Yandex SpeechKit STT — endpoint и параметры в VOICE_PIPELINES.
3. Settings — не хардкодить ключи.
4. **Пустой transcript = ошибка** (`stt_empty`).
5. Retry по VOICE_PIPELINES: 1 retry на timeout/5xx; 401/403 без retry.
6. Mock HTTP в unit-тестах.

## Примеры для проверки

**Unit-тесты (mock HTTP):**

| Mock-ответ API | Ожидание |
| --- | --- |
| `result: "завтра в 15 позвонить врачу"` | `STTResult.text` не пустой |
| `result: ""` | `stt_empty` |
| HTTP 503 → retry → 200 | успех после retry |
| HTTP 401 | ошибка без retry |

**В чате *(опционально, с ключом)*:**

| Голосовое | Ожидание |
| --- | --- |
| Чётко: «Напомни завтра в 15:00 купить хлеб» | Pipeline доходит до parser |
| Шёпот / далеко от микрофона | Может быть `stt_empty` — пользователю «Не расслышал…» |
| Тишина 2–3 сек | `stt_empty` |
| На английском | Может распознаться плохо — не обязательно поддерживать, но не падать |

## Кривые кейсы — учесть

| Ситуация | Ожидание |
| --- | --- |
| Пустой или только пробелы в `result` | `stt_empty`, Task не создаётся |
| Файл повреждён / не opus | Ошибка STT, без partial Task |
| Timeout / 5xx | 1 retry, затем временная ошибка пользователю |
| 401 / 403 (неверный ключ) | Лог без ключа, без бесконечных retry |
| Transcript с лишними пробелами | `.strip()` перед parser |
| Ключ или transcript в логах/snapshots | **Запрещено** |

## Acceptance criteria
